Installing STAFF

In order to install STAFF completely follow the steps below.

Installing Java

First of all, you have to install Java Development Kit (JDK). Please refer Java Documentation for installation instructions.

Please set JAVA_HOME environment variable to refer installed JDK.

If JDK is installed to the "C:\Program Files\Java\jdk1.6.0" directory then JAVA_HOME="C:/Program Files/Java/jdk1.6.0".

Installing STAFF

To install STAFF, just unpack your distribution.

Please add STAFF bin directory to the Path environment variable to be able to run it in from any location.
