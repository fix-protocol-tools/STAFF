fixStop

Description

fixStop task stops FIX initiator, acceptor or executor.

Parameters

Attribute Description Default Required
refid Reference name of the FIX Peer. None Yes
sendLogout Send logout and wait for a logout response before disconnecting or not.
Look at parameter LogoutTimeout of FIX Initiator properties file 		true No

Examples

<target name="stop_all">
	<echo>Stopping FIX Initiator...</echo>
	<fixStop refid="main_initiator"/>
</target>
