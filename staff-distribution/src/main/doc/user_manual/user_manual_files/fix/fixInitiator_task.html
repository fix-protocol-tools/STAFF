fixInitiator

Description

fixInitiator type represents a FIX initiator.

Parameters

Attribute Description Default Required
id Reference name to use FIX Initiator with other tasks. None No
settingsfile FIX Initiator configuration file name. None Yes

Main initiator properties are taken from configuration file. In addition you can set messages control level for the FIX Peer using setControlLevel task.

Examples

	<fixInitiator settingsfile="${fix_common.settings.dir}/FIX44Initiator.properties" id="main_initiator" />
