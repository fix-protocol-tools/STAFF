setControlLevel

Description

Command to switch receive Application/Session Level messages.

Control level can be cahnged at any moment, but take into account, that Receive queue is cleared when setControlLevel is used. So if you want to capture reaction on message sent, change control level before send operator.

Note: if 'session' or 'both' mode is used, system will control all messages, including Logon (A), Heartbeat (0) etc.

Parameters

Attribute Description Default Required
controlLevel Switch receive level messages.Possible values are:
application
session
both		 application Yes

Examples

   <echo>Creating FIX Initiator1...</echo>
   <fixInitiator settingsfile="main_initiator.properties" id="main_initiator" />
   <echo>Change message control level to Session level</echo>
   <setControlLevel refid="main_initiator" controlLevel="session"/>
