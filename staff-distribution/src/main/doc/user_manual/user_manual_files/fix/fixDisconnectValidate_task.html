fixDisconnectValidate

Description

fixDisconnectValidate task checks disconnect signal (if arrived) or wait time left from timeout - (current time - listener install time). This task using together with the fixDisconnectExpect task

Parameters

Attribute Description Default Required
refid Reference name of the fixDisconnectExpect task. None Yes

Examples

    <!-- install listener -->
    <fixDisconnectExpect id="disconnect" refid="main_acceptor" timeout="30"/>
    <!-- do any operation -->
    <fixStop refid="main_acceptor"/>
    <echo>Wait for disconnect...</echo>
    <!-- check disconnect signal (if arrived) or wait time left from timeout - (current time - listener install time) -->
    <fixDisconnectValidate refid="disconnect"/>
