smtpMessage

Description

smtpMessage type represents a SMTP message object.

Parameters

Attribute Description Default Required
id Reference name to use message with other tasks. None No
source Reference name to use defined earlier message. None No

Examples

	<smtpReceive timeout="60">
		<smtpMessage id="msg">
			<field name="From" value="bbb@localhost" />
			<field name="To"   value="aaa@localhost" />    
			<field name="Subject" value="[NOTE] TROIOI:BLP Established" />
			<group name="Data">
				<field name="1" value="N"/>
				<field name="2" value="1"/>
			</group>
		</smtpMessage>
	</smtpReceive>
