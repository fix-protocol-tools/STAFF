TcpStart(deprecated)

Description

TcpStart task starts the TCP session if it was not yet started.

Parameters

Attribute Description Default Required
refid Reference name of the server or client TCP session to be started. None Yes

Examples

      <tcpServer id="TServer"
                 type="transient" port="8765" host="localhost"
                 loginTimeout="30" heartbeatInterval="30" />
      <tcpStart refid="TServer" />
