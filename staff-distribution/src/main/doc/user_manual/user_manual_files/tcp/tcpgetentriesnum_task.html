tcpGetEntriesNum(deprecated)

Description

TcpGetField task extracts number of entities in the specified group from the TCP message and puts its value to the Ant property.

Parameters

Attribute Description Default Required
refid Reference to the TCP message to get group from None Yes
name Field name to get None Yes
gorup Group name to get number of entries from None No
property Property to put field value to The same as field name Yes

In order to apply tcpGetEntriesNum to the message, the massage object should be stored first as described in the example below.

Examples

Get field value

	<tcpReceive refid="ome" sessionID="${OME.SessionID}"  timeout="6000">
		<tcpMessage type="U" id="UserAccountResponse"/>
	</tcpReceive>

	<tcpGetEntriesNum refid="UserAccountResponse" property="prA" group="UserAccounts"/>
	<echo>${prA}</echo>
