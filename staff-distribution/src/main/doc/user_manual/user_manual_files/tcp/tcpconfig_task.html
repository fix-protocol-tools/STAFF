TcpConfig(deprecated)

Description

TcpConfig task is intended for TCP transport environment configuration. As long as Ant and all its tasks are implemented in Java language, TCP transport tasks are implemented using jTcpTransport project. This task should be used if you want to specify custom configuration file and/or library name to use. It is recommended to configure TCP transport before executing other task, otherwise the configuration will take no effect.

Parameters

The attributes for TcpConfig are identical to the jTcpTransport configuration properties.

Attribute Description Default Required
libraryName Name of the DLL/SO library that should be loaded for transport implementation None No
configFile Path to the configuration file that contains jTcpTransport configuration None No

Examples

    <?xml version="1.0" encoding="utf-8"?>
    <project name="test" default="test">
      <taskdef resource="com/btobits/coesfx/automator/antlib.xml"/>
      <dirname property="test.basedir" file="${ant.file}"/>
      <target name="test">
        <tcpConfig
          libraryName="jTcpTransport"
          configFile="${test.basedir}/jTcpTransport.properties" />
      </target>
     </project>
