tcpSimpleReceive(deprecated)

Description

tcpSimpleReceive task receives and compares a data from the TCP session. If no data received task fails. If received data does not correspond specified patter task fails.

Parameters

Attribute Description Default Required
refid Reference name of the TCP session to receive messages from. None Yes
pattern Pattern to compare received data with. Regular expression should be used to specify the pattern. None No
timeout Number of seconds to wait for each message None Yes

Examples

	<tcpSimpleReceive refid="TCPSimpleServer" pattern=".*8=FIX\.4\.2.*35=A.*" timeout="2"/>
