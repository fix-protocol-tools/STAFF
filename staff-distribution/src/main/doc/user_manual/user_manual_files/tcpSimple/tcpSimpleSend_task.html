tcpSimpleSend(deprecated)

Description

tcpSimpleSend task sends portions of data from the file to the TCP session.

Parameters

Attribute Description Default Required
refid Reference name of the TCP session to receive messages from. None Yes
file File containing data to be sent. None Yes
delimiter Regular expression for the delimiter between portions of data in the file. Each portion will be sent separately. '\n' No
repeatDelay Delay between two sends. 0 No

Examples

	<tcpSimpleSend  refid="TCPSimpleClient" file="batches/tcp_fix_logon.txt" delimeter="\n" repeatDelay="0.5"/>
