tcpSimpleClient(deprecated)

Description

tcpSimpleClient type represents simple TCP client.

Parameters

Attribute Description Default Required
id Reference name to use client with other tasks. None Yes
host Host name of the server to connect None Yes
port Port number of the server to connect None Yes

Examples

	<target name="run_all" >
		<mkdir dir="${test.log.dir}" />
		<property file="./settings/tcpSimpleClient.properties" />
		

		<tcpSimpleClient host="${tcpSimpleClient.host}" port="${tcpSimpleClient.port}" id="TCPSimpleClient" />
		<tcpSimpleStart refid="TCPSimpleClient"/>
	</target>
