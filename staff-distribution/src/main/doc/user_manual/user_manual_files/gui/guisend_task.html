GuiSend

Description

GuiSend task sends a set of messages to the connection.

Parameters

Attribute Description Default Required
refid Reference name of the connection to send messages to. None Yes
repeat How many times repeat this operation 1 No
repeatDelay Time to wait between performing operation None No
count How many times send each message 1 No
sendDelay Time to wait between message send operations None No
The GuiSend task should contain a list of messages to be sent.

Examples

      <guiClient id="client" host="localhost" port="8099" />
      <guiStart refid="client" />

      <sleep seconds="1"/>

      <guiSend refid="client">
        <guiMessage type="ADLQ">
          <field name="AC" value="test" />
          <field name="SM" value="EURUSD" />
          <field name="IS" value="0" />
          <field name="UI" value="test" />
        </guiMessage>
      </guiSend>

      <guiStop refid="client" />
