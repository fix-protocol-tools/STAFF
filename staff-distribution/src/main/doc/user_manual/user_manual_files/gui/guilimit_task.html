GuiLimit

Description

GuiLimit task is intended to ensure that no more messages were received by the connection. For example, if the client should not receive any messages during some period of time, you can check it with GuiLimit task. If the message will be received during specified period of time, task fails.

Parameters

Attribute Description Default Required
refid Reference name of the connection to receive messages from. None Yes
timeout Number of seconds to wait for unexpected message 5 No

Examples

    <property name="test.host" value="127.0.0.1" />
    <property name="test.port" value="8099" />

    <guiClient id="client" host="${test.host}" port="${test.port}"/>
    <guiStart refid="client" />

    <guiLimit refid="client" timeout="10" />

    <guiStop refid="client" />
