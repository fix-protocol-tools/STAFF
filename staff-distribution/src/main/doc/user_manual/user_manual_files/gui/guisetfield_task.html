GuiSetField

Description

GuiSetField task assigns value to the message field.

Parameters

Attribute Description Default Required
refid Reference to the message to get field from None Yes
name Field name to get None Yes
value Value to set None Yes

Examples


   <property name="test.host" value="127.0.0.1" />
   <property name="test.port" value="8099" />

   <guiMessage type="CMLI" id="request.login"/>

   <guiSetField name="GT" value="CoesGUI" refid="request.login"/>
   <guiSetField name="BV" value="N/A" refid="request.login"/>
   <guiSetField name="UI" value="test" refid="request.login"/>
   <guiSetField name="PW" value="test" refid="request.login"/>
   <guiSetField name="BP" value="N/A" refid="request.login"/>
   <guiSetField name="BI" value="COESfx" refid="request.login"/>

   <guiClient id="client" host="${test.host}" port="${test.port}"/>

   <guiStart refid="client" />

   <guiSend refid="client">
     <guiMessage source="request.login"/>
   </guiSend>
