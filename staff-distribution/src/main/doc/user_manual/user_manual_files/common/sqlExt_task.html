sqlExt

Description

sqlExt task represents connection to the DB used for the verifyCell task.

Parameters

Attribute Description Default Required
driver Class name of the jdbc driver. None Yes
url Database connection url None Yes
userid Database user name None Yes
password Database password None Yes
query Query which result should be validated. Note the query should contain 'select' statements only. None Yes

Examples

	<sqlExt driver="com.jnetdirect.jsql.JSQLDriver"
		url="jdbc:JSQLConnect://${DBHost}:1433/database=${DBName}"
		userid="${DBUser}"
		password="${DBPassword}"
		query="select * from test_sqlExt order by id">
	        <verifyCell row = "1" field = "str_field" value = "str1"/>
        	<verifyCell row = "2" field = "char_field" value = "2"/>
        	<verifyCell row = "3" field = "date_field" value = "2008-11-27 18:23:43"/>
        	<verifyCell row = "3" field = "decimal_field" value = "33.33"/>
        	<verifyCell row = "4" field = "decimal_field" value = "-44.44"/>
	</sqlExt>
